## Done

- border-width 토큰 추가
- tokens studio 작업 진행

## 1. border-width 토큰 추가

border-width 를 새로 추가했습니다.

- none : 0px
- 1: 1px
- 2: 2px
- 3: 3px
- 4: 4px

3,4 도 사용성이 적을 것 같고, 5이상은 사용하지 않을 것 같다는 판단하에 5가지로 분류하여 추가 작업 진행했습니다.

## 2. tokens studio 작업 진행

tokens studio에 `color`, `space`, `text`, `radius`, `shadow`, `border-width` 항목을 업데이트 했습니다.

미진행 목록: Sementic, transition, z-index, typography

`Sementic`은 결국 실제로 사용하는 디자인 시스템의 핵심 이라고 판단했고, 현재는 여러 토큰들이 디테일하게 추가되고 있는 상황입니다.

0.0.2 version 에서는 꽤 안정화된 배포버전을 원하기 때문에 토큰들이 어느정도 자리 잡히면 `Sementic`을 최종적으로 추가하려고 합니다.

`transition, z-index`는 tokens studio 입력 방법을 물색하고 있으며, 해결하지 못한다면 수기로 타입을 입력해야 할 수도 있을 것 같습니다.
